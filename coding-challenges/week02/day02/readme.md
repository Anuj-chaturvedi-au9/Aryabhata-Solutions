HackerRank : [Loops
](https://www.hackerrank.com/challenges/python-loops/problem?isFullScreen=true) 

```python
if __name__ == '__main__':
    n = int(input())
    i = 0
    while(i < n):
        print(i*i)
        i += 1 #this is equal to i = i + 1
```
